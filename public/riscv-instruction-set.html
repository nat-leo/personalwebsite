RISC-V Base Instruction Set (aka RV64I)

Friday, July 14th, 2023

This is a cheatsheet for RV64I, which does not contain any multiply or divide instructions. RISC-V is split into different formats of instructions, each of which are 32 bits long.

R-Type Instructions

funct7 rs2 rs1 funct3 rd opcode

R-type instructions perform an operation on two operands, like adding two registers, shifting one register left or right a certain number of times specified in a second register, etc. R-type means do something with two values located in two registers.
